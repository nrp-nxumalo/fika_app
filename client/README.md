# Fika Timetables

Fika Timetables is a user-friendly bus timetable viewer for Cape Town commuters. It brings Golden Arrow and MyCiTi route information into one searchable place so people can find the timetable they need without digging through static PDFs or operator pages.

The site is designed for quick everyday lookups: choose an operator, search by route name or number, select a direction, and view service days and stop times in a readable timetable layout. Route pages also make it easier to browse by operator or area, helping commuters discover services that pass through the places they travel from or to.

Fika is especially useful on phones, where timetable information needs to be fast, clear, and easy to scan while travelling. Recently viewed or saved timetables can be stored in the browser for offline access, giving users a practical backup when connectivity is limited.

The goal is simple: make public transport timetable information easier to find, easier to read, and easier to use for real trips.
